Linguagens de Programação

Linguagem Indicado para Tipagem Paradigma Dificuldade Requisitos de Competências Usos Popularidade Comentários
Python Iniciantes Forte/Dinâmica Imperativa/Orientada a Objetos Fácil de aprender, ler e escrever Conhecimento básico em front-end pode facilitar o processo de aprendizado Desenvolvimento web back-end, aplicações desktop, ciência de dados, automação, aplicações de deep learning, machine learning e computação científica. Amada por 66.7% dos desenvolvedores no Stack Overflow Código aberto, altamente escalável, extensivas opções de frameworks, versátil e fácil de aprender. Porém, mais lenta devido ao fato de ser uma linguagem interpretada, não é ideal para desenvolvimento de aplicações móveis
C# Iniciantes e intermediários Forte/Estrita Imperativa/Orientada a Objetos Moderada Conhecimentos básicos em C e em linguagens de programação orientadas a objetos podem facilitar o processo de aprendizagem Aplicações web, aplicações móveis, desenvolvimento de jogos e desenvolvimento de softwares de realidade virtual, aplicações Linux e Mac Amada por 59.7% dos desenvolvedores no Stack Overflow Bom para desenvolvimento Windows e jogos, tempos de compilação e ececução rápidos, mais segura que C e C++ já que não usa ponteiros, grande variedade de frameworks. Porém, tem uma curva de aprendizado inicial um pouco mais difícil que outras linguagens de programação, nãos e traduz bem entre plataformas poque é dependente do ambiente .NET
JavaScript Iniciantes Fraca/Dinâmica Imperativa/Funcional Relativamente fácil Habilidades básicas em programação e familiaridade com HMTL e CSS Desenvolvimento web front-end, desenvolvimento de jogos, aplicações web. Amada por 58.3% dos desenvolvedores no Stack Overflow. Código aberto, o client-side é rápido e não exige compilação, funciona bem com outras linguagens de programação populares, é otíma para o desenvolviemnto web pois permite elevar a experiência do usuário e engajamento no site. Porém apresenta problemas de segurança e desempenho instável entre navegadores
PHP Iniciantes Fraca/Dinâmica Imperativa/Funcional Fácil de aprender Conhecimentos básicos em programação, especialmente CSS e HTMl, podem ajudar no processo de aprendizado Desenvolvimento web, aplicações desktop, programas de computador Amada por 37.3% dos desenvolvedores no Stack Overflow Códgo aberto, as versões PHP 7 em diante são rápidas, bem estabelecida para o desenvolvimento web, suporta POO. Tem menos ferramentas de debugging, criar um site utilzando PHP é uma tarefa mais lenta do que utilzar outras linguagens de programação
Java Iniciantes Forte/Estática Imperativa/Orientada a Objetos Necessita uma curva de aprendizado um pouco maior Conhecimentos básicos em programação Desenvolvimento móvel, desenvolvimento de apps, aplicações desktop, aplicações web, desenvolvimento de jogos, machine learning e computação científica. Amada por 44.1% dos desenvolvedores no Stack Overflow Altamente versáril, possui várias APIs disponíveis para o desenvolvimento de aplicações, segurança aprimorada, compatível entre diferentes plataformas. Porém, possui um tempo de execução lento, consome relativamente bastante memória
Go Iniciantes e intermediários Forte/Estrita Imperativa/Concorrente Fácil de aprender devido ao seu código simples Pode ser mais fácil aprender se você já possui habilidades sólidas em programação C e Java Desenvolvimento web back-end, aplicações web, machine learning, programação de sistemas, big data. Amada por 62.3% dos desenvolvedores no Stack Overflow Código aberto, tempo de execução rápido, segurança aperfeiçoada, excelente suporte para testes e ótimas ferramentas de análise estática. Porém, suporte de bibliotecas limitado, ausência de funções genéricas, não é ideal para desenvolvimento de programas complexos
